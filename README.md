# Daily Exercise + Post-Workout Diet Planner Agent

This is a Streamlit + OpenAI-powered app that:
- Asks about your fitness goal and workout type
- Suggests a daily exercise plan
- Recommends a post-workout meal based on your inputs

## Setup

1. Install dependencies:
```bash
pip install streamlit openai
```

2. Set your OpenAI API key as an environment variable:
```bash
export OPENAI_API_KEY=sk-...
```

3. Run the app:
```bash
streamlit run app.py
```
